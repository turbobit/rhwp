# Task #1006 수행 계획서 — 쪽 테두리 포맷별 분리 + 변환본 cover logo overlap 해소

이슈: [#1006](https://github.com/edwardkim/rhwp/issues/1006)
배경: PR [#956](https://github.com/edwardkim/rhwp/pull/956) 가 본 이슈를 회귀시킴.

## 1. 문제 진단

`samples/hwp3-sample16-hwp5.hwp` (HWP3 → HWP5 변환본) cover page (page 1) 에서 작은 머리말 logo (한국수자원공사) 가 외곽선과 시각적으로 겹침. 한컴 viewer 는 외곽선이 logo 를 포함하도록 표시.

### Root cause (작업지시자 통찰)

HWP3/HWP5/HWPX 의 쪽 테두리 기준 (paper-based vs body-based) 을 단일 `attr bit 0` 해석으로 합쳐 처리한 데 있음. 회귀 사이클:

- **task877**: `paper_based = (attr & 0x01) != 0` — sample16(HWP3) 정합, 시험지(HWP5) 회귀
- **#920**: `paper_based = (attr & 0x01) == 0` — 시험지 정합, sample16 회귀
- **#956**: `paper_based = true` 전역 강제 — HWP5/HWPX 정합, HWP3 회귀
- **#987**: bfid off-by-one 수정 후 attr 존중 복원 → 본 #1006 회귀 (변환본 logo overlap)

### 해결 방향 (작업지시자 지시)

**포맷별/출처별 contract 분리** + **모두 paper-based 통일** (Hancom Office close-up 시각 판정):
- **HWP3**: paper-based (작업지시자 시각 판정 정정 — #987 body-based 는 close-up 비교 부재로 인한 오판단)
- **HWP5/HWPX**: paper-based (#956 정합)
- 공통 렌더러: attr bit 0 단일 해석 X → `PageBorderFill.basis` 명시 필드 직접 사용
- 추가: 머리말 영역 paper-anchored 객체 (logo) 존재 시 outline top clip 건너뜀 (변환본 cover logo overlap 해소)

## 2. 변경 범위

| 영역 | 변경 |
|------|------|
| `src/model/page.rs` | `PageBorderFill::basis: PageBorderBasis` 필드 추가 (`BodyBased` / `PaperBased`) |
| `src/parser/hwp3/mod.rs` | PageBorderFill 생성 시 `basis = BodyBased` 명시 |
| `src/parser/body_text.rs` (HWP5) | PageBorderFill 생성 시 `basis = PaperBased` 명시 |
| `src/parser/hwpx/section.rs` (HWPX) | PageBorderFill 생성 시 `basis = PaperBased` 명시 |
| `src/renderer/layout.rs::build_page_borders` | `paper_based = matches!(pbf.basis, PaperBased)` 로 변경 |
| `src/renderer/layout.rs::build_page_borders` | 머리말 영역 paper-anchored 객체 (logo) 존재 시 outline top clip 건너뛰기 (변환본 cover logo 포함) |
