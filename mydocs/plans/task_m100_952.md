### Issue 1 — 페이지 외곽선 paper/body 잘못 분류

### Issue 2 — sample16 page 18 "나. 주요 과업내용" 본문 다음 페이지 밀림

장기 typeset 결함. 회귀 commit 없음 (9e038d27 에서도 동일 증상). typeset.rs 의 multi-TAC-TopAndBottom shape (caption + picture + caption) 다음 paragraph cursor 가 ~430px 초과 advance. **별도 task** 의 깊은 분석 필요.
