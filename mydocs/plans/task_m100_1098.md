# Task M100-1098 수행 계획서

## 1. 작업 개요

GitHub Issue #1098:

```text
[hwpx] 한컴 변환 exam_kor.hwpx 출력 누락 구현
```

HWP에서 구현된 국어시험 문제지를 한컴 에디터에서 HWPX로 변환한 파일을 rhwp에서 열 때,
원본 HWP 기준으로 출력되어야 할 일부 요소가 출력되지 않는 문제를 처리한다.

## 2. 대상 파일

```text
source HWPX: samples/hwpx/exam_kor.hwpx
oracle HWP:  samples/exam_kor.hwp
```

`samples/hwpx/exam_kor.hwpx`는 이번 작업에서 새로 추가된 샘플이다.

## 3. 목표

```text
1. HWP 원본과 한컴 변환 HWPX의 구조 차이를 확인한다.
2. rhwp-studio에서 HWPX 렌더링 시 출력되지 않는 요소를 특정한다.
3. HWPX parser/IR/renderer 중 누락된 control, shape, image, master page, section element 경로를 구현한다.
4. 기존 HWP `samples/exam_kor.hwp` 렌더링 회귀 없이 HWPX 출력 누락을 해소한다.
```

## 4. Stage 1 계획 — 누락 요소 위치 특정

출력 위치:

```text
output/poc/hwpx/task1098/stage1_exam_kor_gap_trace/
```

작업:

```text
1. HWPX package inventory를 정리한다.
2. HWPX section/masterpage/header/BinData 구조를 추출한다.
3. HWP 원본의 페이지/컨트롤 inventory와 HWPX 변환본의 XML 요소를 비교한다.
4. rhwp-studio에서 출력되지 않는 구간을 페이지 단위로 기록한다.
5. 누락 원인이 parser 미지원인지, IR 보존 누락인지, renderer 미구현인지 분리한다.
```

우선 확인 포인트:

```text
1. 바탕쪽 구현 여부
   - 짝수쪽 적용 바탕쪽
   - 홀수쪽 적용 바탕쪽
   - 마지막쪽 적용 바탕쪽
   - HWP 원본의 바탕쪽 적용 규칙과 HWPX masterpage apply 조건이 같은지 확인

2. 글상자 > 사각형 모서리 곡률 구현 여부
   - HWPX 글상자/사각형의 모서리 둥글기 속성이 parser에서 보존되는지 확인
   - IR shape 모델에 곡률 또는 round-rect 정보가 전달되는지 확인
   - SVG/Canvas renderer가 둥근 사각형으로 출력하는지 확인
```

산출물:

```text
hwpx_inventory.md
hwp_oracle_inventory.md
hwpx_source_trace.md
render_gap_findings.md
masterpage_apply_findings.md
textbox_corner_radius_findings.md
```

## 5. Stage 2 계획 — 최소 구현 후보

Stage 1에서 특정한 누락 요소에 따라 구현 후보를 좁힌다.

예상 후보:

```text
1. HWPX masterpage의 짝수/홀수/마지막쪽 적용 누락
2. 글상자 사각형 모서리 곡률 파싱/IR/렌더링 누락
3. master page 또는 section-level background/shape 누락
4. HWPX shape/object control 파싱 누락
5. 이미지 BinData 참조 또는 clipping/positioning 누락
6. HWPX 전용 control wrapper가 IR로 전달되지 않는 문제
7. HWPX에서만 나타나는 shape 속성의 renderer 미구현
```

## 6. 검증 계획

필수 검증:

```text
cargo fmt --check
cargo check
관련 단위 테스트 또는 regression 테스트
```

시각 판정:

```text
1. `samples/hwpx/exam_kor.hwpx`가 rhwp-studio에서 원본 HWP 기준으로 누락 없이 보이는가
2. `samples/exam_kor.hwp` 기존 렌더링이 회귀하지 않는가
3. 짝수/홀수/마지막 바탕쪽이 원본 HWP와 같은 페이지에 적용되는가
4. 글상자 사각형 모서리 곡률이 원본 HWP와 같은 형태로 출력되는가
5. 문제가 특정 페이지/컨트롤에 한정되어 있다면 해당 페이지를 HWP/HWPX 교차 확인한다
```
